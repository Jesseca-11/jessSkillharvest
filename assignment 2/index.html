COMMANDS THEIR USAGE

1.git init => It is used to Initialize a local Git repository.

2.git clone ssh://git@github.com/[username]/[repository-name].git => It is used to Create a local copy of a remote repository.

3.git status => It is used to Check status.

4.git add [file-name.txt] => It is used to Add a file to the staging area.

6.git commit -m "[commit message]" => It is used to Commit changes.

5.git add -A => It is used to Add all new and changed files to the staging area.

7.git rm -r [file-name.txt] => It is used to Remove a file (or folder).

8.git branch => It is used to List branches (the asterisk denotes the current branch).

9.git branch -a => It is used to List all branches (local and remote).

10.git branch [branch name] => It is used to Create a new branch.

11.git branch -d [branch name] => It is used to Delete a branch.

12.git push origin --delete [branch name] => It is used to Delete a remote branch

13.git checkout -b [branch name] => It is used to Create a new branch and switch to it.

14.git checkout -b [branch name] origin/[branch name] => It is used to Clone a remote branch and switch to it.

15.git branch -m [old branch name] [new branch name] => It is used to Rename a local branch.

16.git checkout [branch name] => It is used to Switch to a branch.

17.git checkout - => It is used to Switch to the branch last checked out.

18.git checkout -- [file-name.txt] => It is used to Discard changes to a file.

19.git merge [branch name] => It is used to Merge a branch into the active branch.

20.git merge [source branch] [target branch] => It is used to Merge a branch into a target branch.

21.git stash => It is used to Stash changes in a dirty working directory.

22.git stash clear => It is used to Remove all stashed entries .

23.git push origin [branch name] => It is used to Push a branch to your remote repository.

24.git push -u origin [branch name] => It is used to Push changes to remote repository (and remember the branch).

25.git push => It is used to Push changes to remote repository (remembered branch).

26.git push origin --delete [branch name] => It is used to Delete a remote branch.

27.git pull => It is used to Update local repository to the newest commit.

28.git pull origin [branch name] => It is used to Pull changes from remote repository.

29.git remote add origin ssh://git@github.com/[username]/[repository-name].git => It is used to Add a remote repository.

30.git remote set-url origin ssh://git@github.com/[username]/[repository-name].git => It is used to Set a repository's origin branch to SSH.

31.git log => It is used to View changes.

32.git log --summary => It is used to View changes (detailed).

33.git log --oneline => It is used to View changes (briefly).

34.git diff [source branch] [target branch] => It is used to Preview changes before merging.
